# Jest

## jest
테스팅 라이브러리.  
프론트 뿐만 아니라 백엔드에서도 쓰임.  
Jest 이전에는 자바스크립트 코드 테스팅을 위해 여러 테스팅 라이브러리를 조합해서 사용해야 했음.  
Jest는 라이브러리 하나만 설치하면 되기 때문에 편리하게 느껴짐.  


## install

1. Jest 라이브러리 설치  
   `npm i -D jest`
2. 스크립트 수정   
```
"scripts": {
   "test": "jest" // Jest로 수정
},
```
3. `npm test` 실행하면 jest 커맨드 실행됨
4. 테스트 코드 작성
