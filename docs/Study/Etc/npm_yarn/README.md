# NPM & Yarn

## Yarn
yarn은 npm과 달리 --save 옵션 주지 않아도 package.json에 자동저장됨.  
* `yarn add` : 현재 폴더의 package.json에 패키지 추가
* `yarn bin` : yarn bin 폴더 경로 표시
* `yarn list` : 패키지 목록 표시
* `yarn remove` : 패키지 삭제
* `yarn upgrade` : 패키지 업데이트
* `yarn upgrade-interactive` : 패키지 버전 선택 가능
