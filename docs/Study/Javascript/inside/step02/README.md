# 자바스크립트 개발 환경

## 웹스톰 설치 및 실행
IDE : 웹스톰 추천.  
[jetbrains](http://www.jetbrains.com/webstorm)에서 받을 수 있음

## 프로젝트 생성 및 프로그래밍

## 테스트 및 디버깅
1. IDE 내 브레이크 포인트 지정 (빨간 점)
2. Run - Debug : 브레이크 포인트에서 멈추게 됨
3. 하단부에 여러 디버깅 정보 노출
4. Step Over, Step Into, Step Out의 명령으로 단계적 실행시키며 변수값 변화를 추적할 수 있음
